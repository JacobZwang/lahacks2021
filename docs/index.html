LA Hacks 2021

By Benjamin Swerdlow, Ethan Hopkins, Jacob Zwang, and Henry Karl Marks

Description

Our project for LA Hacks 2021 is an easy to deploy, adaptable communication tool for anything from calls, meetings, or even conferences.

Tech Stack

Click to Deploy

We created a click to deploy button on Google Cloud that automatically configures the necessary services to have a working tool in approximately two minutes.

WebRTC

We implemented peer to peer video and audio data tracks using WebRTC. The WebRTC handshake was facilitated though our own signaling server, which allowed us to take total control over who people connected to. We used Google's public STUN server to handle our ICE candidates.

Spatial Audio

We implemented the A* Path Finding Algorithm to solve for the shortest path between users, and adjusted sound volume according to user distance to create an immersive feeling. We chose A* over Visualize Dijkstra's because of its higher performance in open spaces.

Canvas

We began this project using a Table, however in order to create a more smooth experience we switched to raw Canvas. We implemented our own pan, zoom, and move effects.

Big Table

We chose Big Table for logs because of its easy deployment, close integration with other Google Cloud services, and Wide Column Structure.

Use Cases

Our easy deployment combined with spatial audio technology makes any video call more interesting and productive.
